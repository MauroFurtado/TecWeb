Meus Projetos

Esta seção apresenta alguns dos projetos que desenvolvi ao longo dos meus estudos e trabalhos independentes. Cada projeto reflete o meu crescimento como estudante de Engenharia de Computação, onde apliquei tecnologias e conceitos práticos. Aqui, você encontrará uma breve descrição de cada projeto, incluindo o objetivo, as ferramentas utilizadas e os principais resultados obtidos.

  1. Rede Neural perceptron

    2. Neste projeto foi criado uma rede Perceptron em Octave para realizar a classificação binária de pontos em um plano bidimensional. O usuário fornece um conjunto de vetores de entrada com rótulos (0 ou 1), e o Perceptron é treinado para ajustar seus pesos com base nesses dados. Após o treinamento, o código exibe os pesos finais e plota os pontos de entrada com uma linha de decisão que separa as duas classes. O objetivo é que o Perceptron aprenda a distinguir entre as classes com base nas coordenadas fornecidas, adaptando os pesos para minimizar erros de classificação.

  2. RBF

    3. Nesse projeto é implementada uma rede de base radial (RBF) para classificar dados em duas classes. Ele calcula a saída das RBFs para cada ponto de entrada, ajusta os pesos para minimizar o erro de classificação e, em seguida, realiza previsões. A precisão do modelo é calculada, e uma superfície de decisão é gerada e plotada, mostrando visualmente o limite entre as classes.

  3. MLP

    4. Neste projeto utilizei uma Rede Neural Perceptron Multicamadas (MLP) para classificar dados em três categorias. Primeiramente, os dados são carregados e processados, incluindo a transformação de rótulos categóricos em uma matriz de classificação. A rede neural é treinada em um conjunto de dados com 70% para treino e 30% para teste, repetindo o processo 10 vezes para calcular a acurácia média. Cada execução embaralha os dados para garantir robustez nos resultados. A rede neural, com uma camada oculta de 6 neurônios e uma camada de saída de 3 neurônios, atinge uma acurácia média após as execuções, indicando sua capacidade de generalização para classificação.
